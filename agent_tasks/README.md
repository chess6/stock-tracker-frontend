# Agent Tasks

Runtime task queue for the autonomous repair orchestrator.

| Folder | Contents |
|--------|----------|
| `pending/` | Discovered issues awaiting repair |
| `active/` | Task currently being worked |
| `completed/` | QA-passed tasks (with commit metadata) |
| `failed/` | Budget-exhausted or ambiguous tasks |

Task files are JSON (`{task_id}.json`). Also see `.cursor/agent_tasks/{id}.md` for Cursor repair briefs.

`health-scan-report.json` is regenerated by `e2e/health-scan.spec.js`.
